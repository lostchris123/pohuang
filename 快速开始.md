## 六、部署指南

### 构建生产版本
```bash
npm run build
```

### 预览构建结果
```bash
npm run preview
```

### 部署到服务器
将 `dist/` 目录上传到您的服务器即可

### 推荐部署平台
1. **Vercel** - 一键部署，自动CDN
2. **Netlify** - 免费托管，CI/CD
3. **阿里云OSS** - 国内加速
4. **腾讯云COS** - 快速访问
