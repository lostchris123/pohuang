# 杭州破荒人工智能科技有限公司官网 - 开发说明

## 🚀 快速启动指南

### 1. 安装依赖
```bash
cd /home/gem/.openclaw/workspace/pohuang-ai-website
npm install
# 或使用yarn
yarn install
# 或使用pnpm
pnpm install
```

### 2. 开发模式
```bash
npm run dev
```
然后打开浏览器访问: http://localhost:3000

### 3. 生产构建
```bash
npm run build
```
构建结果在 `dist/` 目录

### 4. 预览构建结果
```bash
npm run preview
```

## 📁 项目结构说明

```
pohuang-ai-website/
├── src/
│   ├── components/     # 可复用组件

│   │   ├── Footer.tsx     - 网站页脚
│   │   ├── HeroCarousel.tsx - 首页轮播图
│   │   └── ProductCard.tsx  - 产品卡片组件

│   │   ├── HomePage.tsx    - 首页
│   │   ├── ProductsPage.tsx - 产品中心
│   │   ├── AboutPage.tsx   - 企业简介
│   │   └── ContactPage.tsx - 联系我们
│   ├── styles/        # 样式文件
│   │   └── globals.css - 全局Tailwind样式
│   ├── utils/         # 工具函数
│   │   ├── constants.ts  - 常量数据
│   │   └── helpers.ts    - 工具函数
│   ├── App.tsx        - 应用路由配置
│   └── main.tsx       - 应用入口
├── public/            # 静态资源
├── package.json       - 项目依赖配置
├── tailwind.config.js - Tailwind CSS配置
├── vite.config.ts     - Vite构建配置
├── README.md          - 项目文档
└── 项目总结.md        - 项目开发总结
```

## 🛠️ 技术栈详情

### 核心框架
- **React 18** - 前端UI库
- **TypeScript** - 类型安全
- **React Router 6** - 路由管理

### 样式系统
- **Tailwind CSS 3.4** - 原子化CSS框架
- **Framer Motion** - 动画库
- **Lucide React** - 图标库

### 构建工具
- **Vite 5** - 极速构建工具
- **PostCSS** - CSS处理
- **Autoprefixer** - 浏览器前缀

### 开发工具
- **ESLint** - 代码检查
- **Prettier** - 代码格式化
- **TypeScript Compiler** - 类型检查

## 🚀 部署指南

### 本地开发
```bash
npm run dev
# 访问 http://localhost:3000
```

### 生产部署
```bash
# 构建
npm run build

# 预览构建结果
npm run preview

# 部署到服务器
# 将dist目录上传到服务器
```

### 部署平台推荐
1. **Vercel** - 一键部署，自动CDN
2. **Netlify** - 免费托管，CI/CD
3. **阿里云OSS** - 静态网站托管
4. **腾讯云COS** - 国内加速
